# 🎮 Top-Down Shooter

A simple 2D Top-Down Shooter game built using Unity as part of the **20 Games Making Challenge**.

---

## 🚀 Game Overview

In this game, the player controls a character from a top-down view and must survive by shooting enemies while avoiding damage.

---

## 🎯 Features

- 🧍 Player movement using WASD / Arrow keys  
- 🔫 Shooting system using keyboard input  
- 💥 Bullet collision detection  
- 👾 Enemies that chase the player  
- ❌ Enemy destruction when hit  
- ❤️ Player health system  
- 📈 Score system  
- 🧾 UI displaying score and health  
- 🔁 Game restart after Game Over  

---

## 🛠️ Engine Used

- Unity (2D Core)

---

## 🧠 What I Learned

- Player movement using Rigidbody2D  
- Implementing shooting mechanics  
- Collision detection using Unity Physics  
- Enemy AI movement (chasing player)  
- Managing game state (score, health, restart)  
- UI integration in Unity  

---

## ⚠️ Challenges Faced

- Bullet collision not detecting properly  
- Score not updating due to script connection issues  
- Player disappearing due to physics conflicts  
- Health decreasing instantly due to multiple collisions  
- Fixing proper Rigidbody and Collider configurations

## ▶️ How to Play

- Move → **WASD / Arrow Keys**  
- Shoot → **Spacebar**  
- Avoid enemies and survive as long as possible  

---

## 📦 Build / Run Instructions

1. Open project in Unity  
2. Open the main scene  
3. Click **Play**
